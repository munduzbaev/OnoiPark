# ParkEasy - Умная система управления парковкой

Полнофункциональное приложение для управления парковкой с бронированием мест, QR-сканированием, системой тарификации и профилем пользователя.

## Основные функции

### 1. Регистрация и Профиль

- **Вход/Регистрация**: Использует госномер как логин и номер телефона как пароль
- **Профиль пользователя**: Настройки уведомлений, история парковок, способы оплаты
- **Политика конфиденциальности**: Доступна прямо из профиля
- **Техническая поддержка**: Быстрая связь с оператором

### 2. Карта и Поиск Парковки

- **Интерактивная карта**: Отображение всех доступных парковок
- **Реал-тайм статус**: Количество свободных/занятых мест для каждой парковки
- **Цветовая индикация**: 
  - 🟢 Зелёный: >50% мест свободно
  - 🟠 Оранжевый: 20-50% мест свободно
  - 🔴 Красный: <20% мест свободно
- **Детальная информация**: Адрес, тариф, особенности парковки

### 3. Схема Парковки

- **120 парковочных мест**: Визуализация всех мест на парковке
- **Статусы мест**:
  - ✅ Свободно (зелёный)
  - 🔒 Забронировано (оранжевый)
  - ❌ Занято (красный)
- **Выбор места**: Интерактивный выбор свободного места
- **Организация по рядам**: Удобная навигация (Ряд A, B, C и т.д.)

### 4. Бронирование и Парковка

#### Бронирование места
- Выбор свободного места на схеме
- Резервация на 15 минут
- Уведомление о подтверждении бронирования

#### Начало парковки
- QR-сканер для подтверждения въезда
- Автоматическая фиксация времени начала
- Начало бесплатного периода (1 час)

### 5. Тарификация

- **Бесплатный период**: Первый час парковки бесплатный
- **Почасовая оплата**: После бесплатного часа - 100 ₽/час
- **Реал-тайм отображение**: 
  - Таймер бесплатного времени
  - Текущая стоимость парковки
  - Общее время парковки
- **Автоматический расчёт**: Посекундное обновление стоимости

### 6. Активная Сессия

- **Живой таймер**: Обратный отсчёт бесплатного времени
- **Визуальные предупреждения**: 
  - За 15 минут до окончания бесплатного времени
  - Анимированные уведомления
- **Информация о парковке**:
  - Номер места
  - Адрес парковки
  - Время начала
  - Текущая стоимость
- **Действия**: Навигация, продление, завершение

### 7. Уведомления

Настраиваемые пуш-уведомления:
- ✅ Подтверждение бронирования
- ⏰ Окончание бесплатного периода (за 10-15 минут)
- ⏳ Напоминание о завершении парковки (за 5-10 минут)
- 💳 Подтверждение оплаты

### 8. История Парковок

- Детальный список всех завершённых сессий
- Информация по каждой сессии:
  - Дата и время
  - Парковка и место
  - Продолжительность
  - Стоимость
  - Госномер автомобиля
- Экспорт истории

### 9. Оплата

- Привязка банковских карт
- Автоматическая оплата при завершении
- Поддержка различных платёжных систем
- Безопасные транзакции

## Технологии

- **Frontend**: React, TypeScript, Tailwind CSS
- **UI Компоненты**: shadcn/ui
- **Backend**: Supabase Edge Functions, Hono
- **База данных**: Supabase (KV Store)
- **Аутентификация**: Supabase Auth
- **Уведомления**: Sonner (toast notifications)

## Структура данных

### Парковки
- 3 парковки по умолчанию (университетская, ресторан, ТЦ)
- Каждая с 60-200 местами
- Реал-тайм обновление статуса

### Места парковки
- Уникальный номер (1-120)
- Статус (available/booked/occupied)
- Привязка к парковке

### Пользователи
- Госномер (логин)
- Номер телефона (пароль)
- Имя
- Настройки уведомлений
- Способы оплаты

### Сессии
- Время начала/конца
- Парковка и место
- Стоимость
- Статус (active/completed)

## Безопасность

- Шифрование данных
- Безопасное хранение паролей через Supabase Auth
- Защита API endpoints
- Приватные bucket для файлов

## Использование

1. **Регистрация**: Введите госномер и телефон
2. **Поиск парковки**: Выберите парковку на карте
3. **Выбор места**: Просмотрите схему и выберите свободное место
4. **Бронирование**: Забронируйте место на 15 минут
5. **Въезд**: Отсканируйте QR-код на въезде
6. **Парковка**: Первый час бесплатно, далее 100 ₽/час
7. **Завершение**: Завершите сессию и оплатите (если требуется)
